워크샵 환경은 서울리전(ap-northeast-2) 기본, US리전도 사용 가능.

**Why:** LGE MS사업본부 워크스페이스가 서울리전이지만 일부 기능(Lakebase 등)은 US리전에서만 가능할 수 있음.

**How to apply:** 
- Lakebase 관련 내용을 "선택" 사항이 아닌 **사용 가능 옵션**으로 포함
- 에이전트 세션 관리, OLTP 등 Lakebase 활용 시나리오를 기본 가이드에 포함
- 서울리전에서 Lakebase 안 될 경우 US리전 대안 안내
